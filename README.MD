# 22qq's Custom Utilities Mod
## Features:
* PopCounter (Counts Pops, Credit to Kisrr)
* NoHurtCam
* NoFireOverlay
* Crystal Optimizer
* NameTag Visibility

# Controls (you can change it)
* F8 - Toggles popcounter
* F9 - Resets popcounter
* 8  - NoHurtCam
* 9  - NameTag Visibility
* 0  - Crystal Optimizer
